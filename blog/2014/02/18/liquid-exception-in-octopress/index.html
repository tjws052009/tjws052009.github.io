Liquid Exception in Octopress

octopressを使ったブログを更新してる時に下記のようなエラーに出くわした。

どうもコードをコピーして来た時に、 {{ }} をくっつけて書いたのが気にくわないらしい。

そのため、octopressでは二重中カッコを書く時は対応方法がふたつあるらしい

  • 1) 2つの中カッコの間にスペースを入れる { {
  • 2) {% raw %} {{ {% endraw %} のようにで文字列を囲む
